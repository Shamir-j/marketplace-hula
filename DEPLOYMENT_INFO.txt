                      TROUBLESHOOTING TIPS
================================================================================

Issue: Pages showing 404 errors
Fix:   - Ensure all files extracted to public_html
       - Check file permissions (644 for files, 755 for dirs)
       - Verify _next folder exists

Issue: Styling/CSS not loading
Fix:   - Check _next/static/chunks folder exists
       - Clear browser cache (Ctrl+Shift+Del)
       - Check browser console for errors

Issue: Images not displaying
Fix:   - Verify public folder extracted correctly
       - Check image file permissions
       - Check image paths in HTML

Issue: Slow loading
Fix:   - Enable GZIP in cPanel
       - Enable browser caching
       - Consider using CDN
       - Check server resources
